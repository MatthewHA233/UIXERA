# UIXERA

> 消灭 AI 生成的廉价感 — 前端工程师的电影级 UI 提示词武器库

## 这是什么

一套精心策划的 **AI 提示词系统**，专门用于指导 AI（Claude / GPT / Cursor 等）生成**高保真、电影感、像素级完美**的落地页和 UI 界面。

核心理念：**一句话触发高级审美**，比堆砌提示词微调效果好得多。

## 为什么需要它

用 AI 写前端最大的问题不是"能不能跑"，而是**看起来像 AI 做的**。UIXERA 通过精确的美学预设系统（配色、字体、图片风格、文案模式）直接把 AI 输出拉到作品集级别。

## 使用方式

1. 将提示词文件内容粘贴到你的 AI 工具（Claude / GPT / Cursor）的系统提示或对话中
2. AI 会自动提出 4 个问题：品牌名、美学方向、价值主张、行动号召
3. 选择一个预设，AI 直接输出完整的 React + Tailwind + GSAP 落地页代码

## 技术栈

提示词生成的项目默认使用：

- React 19 + Vite
- Tailwind CSS v3.4
- GSAP 3 + ScrollTrigger
- Lucide React 图标
- Google Fonts
